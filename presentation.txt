

2-1 - Situation actuelle
Les ré-écoutes sont réalisées par les superviseurs, avec des grilles au format Excel.

•	stockage des grilles d'écoute aléatoire, chaque superviseur a sa propre gestion (sur PC, dans la GED dans différents répertoires…)
•	absence d'outils de planification et de suivi des réalisations
•	absence d'états de synthèse à tous les niveaux (individuels, équipe, atelier)


2-2 - Objectif du projet
Les objectifs sont multiples :
•	uniformiser les outils utilisés par les superviseurs
•	améliorer le suivi des ré-écoutes en les automatisant
•	centraliser les grilles et les états de synthèse
•	se mettre en conformité par rapport à la norme NF345


Les fonctionnalités de l'outils
- Interface d'écoute
- Interface de notation 
	commentaire obligatoire
	commentaire générale obligatoire
	objectif obligatoire	

- Interface de suivi (visualisation / modification / suppression)
- Interface pour la synthèse
- Interface pour les indicateurs NF

Remarques : 
- 1 TLC ne vera que ses grilles
